# Kindle Package Manager
Because everything else kinda sucks, ngl.

## Goals
- Easy to use CLI
- Lightweight, understandable codebase
- Repositories hostable on static hosting sites such as Github pages
- Packages easy for anyone to make

## How to Install
1. Install the KMC Universal Hotfix
2. Use the search-bar command `;kpm` to invoke `KPM`
