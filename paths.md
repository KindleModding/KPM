# Paths
KPM stores its package database in

- `/mnt/us/kmc/kpm/kpm.db` - This file stores internal KPM data
- `/mnt/us/kmc/kpm/packages/` - This folder contains KPM packages
- `/mnt/us/kmc/kpm/tmp/` - This folder is used to store temporary files such as package downloads
- `/mnt/us/kmc/kpm/cache/` - This folder is used to store cached data (such as screenshots)
