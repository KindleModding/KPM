\hypertarget{md_README_autotoc_md11}{}\doxysubsection{\texorpdfstring{hooks}{hooks}}\label{md_README_autotoc_md11}
The following scripts are all optional and can be placed at the {\itshape root} of the package will be called under certain events\+: ~\newline

\begin{DoxyItemize}
\item install.\+sh -\/ called once the package has been downloaded and extracted
\item launch.\+sh -\/ called if launched from a launcher (NOTE\+: Apps are expected to manage THEIR OWN SCRIPTLETS via {\ttfamily install.\+sh} \& {\ttfamily uninstall.\+sh})
\item uninstall.\+sh -\/ called before the package is deleted
\end{DoxyItemize}

All of these scripts will be executed via {\ttfamily sh} NOT {\ttfamily bash}\hypertarget{md_README_autotoc_md12}{}\doxysubsection{\texorpdfstring{Installation Procedure}{Installation Procedure}}\label{md_README_autotoc_md12}
App installation is handled by \doxylink{classKPM}{KPM} as follows\+:
\begin{DoxyEnumerate}
\item App is downloaded to {\ttfamily /mnt/us/kpm/tmp}
\item App is extracted to {\ttfamily /mnt/us/kpm/packages}
\item App is added to the package index database 
\end{DoxyEnumerate}
